title UC 8 - Visualiser la liste des mutualistes suspendus

Acteurs: Administrateur
Lifelines:
  Administrateur
  Interface Administration (UI Admin)
  Système de Gestion Adhérents (Backend/Système)
  Base de Données (DB)

Flux:
  Administrateur -> UI Admin: Accéder section gestion adhérents
  Administrateur -> UI Admin: Cliquer sur lien/option "Liste Mutualistes Suspendus" ou appliquer filtre

  UI Admin -> Système de Gestion Adhérents: Demander liste mutualistes statut "Suspendu"

  Système de Gestion Adhérents -> DB: Interroger base pour mutualistes statut "Suspendu"
  DB --> Système de Gestion Adhérents: Retourner liste des mutualistes suspendus (ou liste vide)

  Opt [Aucun mutualiste suspendu (Scénario Alternatif A1)]
    Système de Gestion Adhérents --> UI Admin: Indiquer liste vide
    UI Admin --> Administrateur: Afficher message "Aucun mutualiste suspendu" ou liste vide

  Else [Liste de mutualistes suspendus non vide]
    Système de Gestion Adhérents --> UI Admin: Envoyer liste des mutualistes suspendus
    UI Admin --> Administrateur: Afficher la liste paginée/filtrable des mutualistes suspendus avec infos clés

  Alt [Erreur Système lors récupération (Scénario d'Exception E1)]
    Système de Gestion Adhérents -> Système de Gestion Adhérents: Enregistrer log d'erreur technique
    Système de Gestion Adhérents --> UI Admin: Indiquer Échec Technique
    UI Admin --> Administrateur: Afficher message d'erreur générique
