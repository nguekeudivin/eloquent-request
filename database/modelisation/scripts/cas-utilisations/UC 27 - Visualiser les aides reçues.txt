title UC 27 - Visualiser les aides reçues

Acteurs: Mutualiste
Lifelines:
  Mutualiste
  Interface Utilisateur (UI)
  Système de Gestion Aides (Backend/Système)
  Base de Données (DB)

Flux:
  Mutualiste -> UI: Se connecter à son espace personnel (UC 1)
  Mutualiste -> UI: Naviguer vers section "Mes Aides" / "Aides Reçues"

  UI -> Système de Gestion Aides: Demander liste des aides pour le mutualiste [ID Mutualiste]

  Système de Gestion Aides -> DB: Interroger base pour aides associées au mutualiste [ID Mutualiste]
  DB --> Système de Gestion Aides: Retourner liste des aides (ou liste vide)

  Opt [Aucune aide enregistrée pour ce mutualiste (Alt A1)]
    Système de Gestion Aides --> UI: Indiquer liste vide
    UI --> Mutualiste: Afficher message "Aucune aide enregistrée" ou liste vide

  Else [Liste d'aides non vide]
    Système de Gestion Aides --> UI: Envoyer liste des aides (avec infos clés: Date, Type, Montant/Description, Statut)
    UI --> Mutualiste: Afficher le tableau/la liste des aides reçues

  Alt [Erreur Système lors récupération (Exc E1)]
    Système de Gestion Aides -> Système de Gestion Aides: Enregistrer log d'erreur technique
    Système de Gestion Aides --> UI: Indiquer Échec Technique
    UI --> Mutualiste: Afficher message d'erreur générique ("Impossible de charger la liste des aides")
