title UC 17 - Visualiser l encours d un prêt

Acteurs: Administrateur
Lifelines:
  Administrateur
  Interface Administration (UI Admin)
  Système de Gestion Prêts (Backend/Système)
  Base de Données (DB)

Flux:
  Administrateur -> UI Admin: Accéder section gestion prêts
  Administrateur -> UI Admin: Rechercher/Sélectionner un prêt [ID Prêt]

  UI Admin -> Système de Gestion Prêts: Demander informations et encours prêt [ID Prêt]

  Système de Gestion Prêts -> DB: Récupérer données prêt, historique paiements, échéancier [ID Prêt]

  Alt [Prêt trouvé et accessible]
    DB --> Système de Gestion Prêts: Retourner données prêt
    Système de Gestion Prêts -> Système de Gestion Prêts: Calculer encours restant dû
    Système de Gestion Prêts --> UI Admin: Envoyer informations résumées et encours calculé
    UI Admin --> Administrateur: Afficher vue résumée du prêt avec encours et détails clés

  Alt [Prêt introuvable/non accessible (Alt A1)]
    DB --> Système de Gestion Prêts: Indiquer prêt non trouvé/non accessible
    Système de Gestion Prêts --> UI Admin: Indiquer erreur (prêt introuvable/non accessible)
    UI Admin --> Administrateur: Afficher message d'erreur

  Alt [Erreur Système lors récupération/calcul (Exc E1)]
    Système de Gestion Prêts -> Système de Gestion Prêts: Enregistrer log d'erreur technique
    Système de Gestion Prêts --> UI Admin: Indiquer Échec Technique
    UI Admin --> Administrateur: Afficher message d'erreur générique
