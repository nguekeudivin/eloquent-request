title UC 26 - Visualiser l ensemble des aides accordées

Acteurs: Administrateur
Lifelines:
  Administrateur
  Interface Administration (UI Admin)
  Système de Gestion Aides (Backend/Système)
  Base de Données (DB)

Flux:
  Administrateur -> UI Admin: Accéder section consultation aides
  Administrateur -> UI Admin: Sélectionner option "Liste de toutes les aides"

  UI Admin -> Système de Gestion Aides: Demander liste complète des aides
  Système de Gestion Aides -> DB: Interroger base pour toutes les aides enregistrées

  Alt [Aides trouvées]
    DB --> Système de Gestion Aides: Retourner liste des aides
    Système de Gestion Aides --> UI Admin: Envoyer liste des aides (avec infos clés: Type, Montant/Description, Bénéficiaire, Statut)
    UI Admin --> Administrateur: Afficher le tableau/la liste de toutes les aides

  Alt [Aucune aide enregistrée (Alt A1)]
    DB --> Système de Gestion Aides: Indiquer liste vide
    Système de Gestion Aides --> UI Admin: Indiquer liste vide
    UI Admin --> Administrateur: Afficher message "Aucune aide enregistrée" ou liste vide

  Alt [Erreur Système lors récupération (Exc E1)]
    Système de Gestion Aides -> Système de Gestion Aides: Enregistrer log d'erreur technique
    Système de Gestion Aides --> UI Admin: Indiquer Échec Technique
    UI Admin --> Administrateur: Afficher message d'erreur générique ("Impossible de charger la liste des aides")
